Etiqueta <code></code>

La etiqueta HTML <code> se utiliza para mostrar texto que debe ser interpretado como código informático. Este código puede incluir etiquetas HTML, CSS, JavaScript u otros lenguajes de programación. La etiqueta <code> es útil para resaltar el código de un documento HTML y hacer que sea más fácil de leer y entender

Ejemplo

JavaScript: console. log("Hello world!");

Python: print('Hola mundo')
